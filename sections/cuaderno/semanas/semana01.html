Resumen y definiciones

Esta semana aprendí los conceptos fundamentales de las tecnologías web, diferenciando entre sistemas, aplicaciones, sitios y páginas web. Comprendí cómo funciona la web a nivel de protocolos (DNS, TCP/IP, HTTP) y los distintos roles en el desarrollo web. Además, me familiaricé con Visual Studio Code como entorno de desarrollo y la importancia de un portafolio educativo.

Reflexión

¿Qué aprendí? Esta semana comprendí los fundamentos de las tecnologías web, la diferencia entre sistemas, aplicaciones, sitios y páginas web, así como el funcionamiento básico de la web (DNS, TCP/IP, HTTP) y los roles en el desarrollo web. Además, aprendí a utilizar Visual Studio Code como herramienta esencial para el desarrollo y gestión de proyectos web.

¿Cómo aprendí? Aprendí a través de la exposición del sílabo, la explicación teórica de los conceptos y la realización de una prueba de diagnóstico. La práctica en laboratorio con Visual Studio Code y la colaboración con mis compañeros me permitieron afianzar los conocimientos y familiarizarme con el entorno de desarrollo.
