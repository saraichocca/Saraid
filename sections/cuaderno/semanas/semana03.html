¿Qué aprendí? Aprendí a desarrollar interfaces web modernas utilizando las librerías CSS Bootstrap y Tailwind, explorando componentes, layouts y utilidades para crear diseños atractivos y funcionales.

¿Cómo aprendí? Aprendí mediante la práctica en laboratorio, el análisis de documentación oficial y la aplicación de los conceptos en el desarrollo de un sitio web, lo que me permitió comparar y elegir la mejor herramienta para cada necesidad.
